# 🦇 Vampire Bio Muter Bot

A powerful Telegram moderation bot that automatically detects users with promotional links in their Telegram bio and removes their messages instantly.

## ✨ Features

* Automatically deletes messages from users with bio links
* Detects Telegram links, usernames, and promotional URLs
* Ignores group admins and approved users
* Approve trusted users manually
* Clean, fast, and secure moderation
* Easy deployment on Heroku, Render, or VPS

## 🚀 Commands

* `/start` — Start the bot
* `/help` — Show help menu
* `/approve` — Approve a replied user
* `/unapprove` — Remove approval from a replied user
* `/approved` — Show total approved users

## 🛡️ How It Works

1. User sends a message in your group.
2. Bot checks whether the user is an admin or approved.
3. Bot scans the user's Telegram bio.
4. If a promotional link or username is detected, the message is deleted.
5. User receives a warning to remove the bio link.
